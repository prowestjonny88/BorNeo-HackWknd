# Pasar Memory

Pasar Memory is a mobile-first business memory app for informal merchants and hawkers. It helps users reconstruct a trusted end-of-day ledger from mixed signals such as payment screenshots, voice recap, counted cash, optional live taps, and export files.

## Brief Summary

This project is built for merchants who need a fast way to answer a simple but important question: "How much did I actually sell today?" Instead of requiring full POS-style logging, Pasar Memory combines evidence the merchant already has and turns it into a clearer, more trustworthy daily sales record.
